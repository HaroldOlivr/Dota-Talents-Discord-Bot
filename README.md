# Bot Commands:
##### All commands must begin with the prefix: `::`
### Miscellaneous:
| **Key** |                                       **Value**                                               |
|-------------|----------------------------------------------------------------------------------------------------|
| `::map` | Pulls up the new Dota 2 map that displays structure positions and neutral creeps pull/stack times. |
